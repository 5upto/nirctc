Steps to run the NIRCTC application:

- Install Python if not already (Make sure to add Python to PATH variables)
- Open NIRCTC directory
- run "railway_venv\Scripts\activate" to activate the virtual environment
- run "pip install -r requirements.txt" to install the dependencies
- run "python __main__.py" to run the application
